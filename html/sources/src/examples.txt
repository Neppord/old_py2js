Examples
========

All examples are in the ``examples`` directory.

Game of Life
------------

`gol.html <../_static/examples/gol.html>`_ uses the HTML5 Canvas to draw
the game of life. Source:
`examples/gol.py <http://github.com/mattpap/py2js/blob/master/examples/gol.py>`_

ExtJS demo
----------

`ui.html <../_static/examples/ui.html>`_ shows how to call the
`ExtJS <http://extjs.com/>`_ library from Python to create a nice GUI in the
browser. Source:
`examples/ui.py <http://github.com/mattpap/py2js/blob/master/examples/ui.py>`_
